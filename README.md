# Liluama Colombia

> Suplemento para la salud urinaria femenina con graviola, noni y plantas medicinales colombianas.

**[Visitar liluama.co](https://liluama.co)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Liluama Colombia](https://liluama.co), un recurso de salud basado en evidencia científica enfocado en **Salud Urinaria Femenina**.

### Ingredientes clave

Graviola (*Annona muricata*), noni (*Morinda citrifolia*), flor blanca, D-manosa, probióticos

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
